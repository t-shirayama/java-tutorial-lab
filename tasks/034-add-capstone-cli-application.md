# タスク: 総合演習として小さなCLIアプリを追加する

## 背景

25章まで学んだ内容を、最後に1つの小さなアプリで統合できると、学習体験が強くなります。

## 対象

- 新規章または付録
- `README.md`
- 新規 `docs/<chapter>-<slug>/README.md`
- 新規 `examples/`

## 修正内容

- 学習メモ管理アプリ、またはタスク管理CLIを追加する。
- `record`、`enum`、`List`、`Map`、Stream、Optional、ファイル入出力、例外処理、JUnit、ログを使う。
- Step 1〜Step 8のように段階的に作れる構成にする。
- Mavenでアプリ実行とテスト実行ができるようにする。

## 完了条件

- 読者が「Javaで小さなアプリを作れた」と感じられる。
- 既存章の知識を自然に復習できる。
- Docker実行コマンドとローカルMaven実行コマンドがある。
