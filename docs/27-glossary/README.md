# 27章 用語集

この章は、Java学習中に出てくる用語を短く復習するためのページです。分からない言葉に出会ったら、該当する章へ戻る入口として使ってください。

## この章でできるようになること

この章を終えると、次のことができるようになります。

- Java基礎でよく出る用語を短く説明できる
- 分からない用語から関連章へ戻れる
- JDK、JVM、Maven、クラス、例外、Streamなどの位置づけを整理できる

## オブジェクト指向

| 用語 | 意味 | 関連章 |
| --- | --- | --- |
| クラス | オブジェクトの設計図です。フィールドやメソッドを定義します。 | [6章](../06-classes/) |
| インスタンス | クラスから作られた実体です。 | [6章](../06-classes/) |
| フィールド | オブジェクトが持つ値です。 | [6章](../06-classes/) |
| メソッド | オブジェクトやクラスが持つ処理です。 | [6章](../06-classes/) |

## よく使う構文とAPI

| 用語 | 意味 | 関連章 |
| --- | --- | --- |
| 例外 | 処理を続けられない状況を呼び出し元へ伝える仕組みです。 | [14章](../14-exceptions/) |
